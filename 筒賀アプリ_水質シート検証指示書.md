### 2. 水質管理報告シート 転記実装の検証

水質管理報告は **GAS が複数のソースから値を集めて書き込む複合転記** が必要。最も複雑なロジックなので入念に検証する。

#### 2.1 構造確認
- [ ] 前半 r8-r38 / 後半 r49-r79 の 31日連続構造
- [ ] **後半行 = 前半行 + 41** の関係で実装されている
- [ ] **平日のみ書き込み**（土日は空欄）が実装されている
- [ ] **月毎に平日マッピングが動的に計算**される（3月固定実装になっていないか）
   - 3月: 22平日
   - 4月: 22平日（曜日配置で変動）
   - 2月: 19-20平日
   - 月の日数と1日の曜日から平日行を動的に決定

#### 2.2 行計算の関数
以下の関数または同等のロジックが実装されているか:

```javascript
function getWQRRows(dayOfMonth) {
  const front = 7 + dayOfMonth;  // 例: day=2 → r9
  return { frontRow: front, backRow: front + 41 };
}
```
