## 1. 概要

筒賀水質管理センター（広島県）の週次点検データをスマートフォンから入力できるWebアプリを、加計浄化センター アプリを雛形にして構築する。

### 完成イメージ

- スマートフォン最適化されたPWA（GitHub Pages公開）
- Googleスプレッドシート（SS-D）にデータ蓄積
- Excel出力時はテンプレファイルに値を埋め込んで.xlsxダウンロード
- オフライン対応、前日コピー、Undo機能あり

### 技術構成（加計と同じ）

- **フロントエンド**: HTML/CSS/JS（PWA、SheetJS同梱）
- **バックエンド**: Google Apps Script（doPost/doGet）
- **DB**: Googleスプレッドシート
- **公開**: GitHub Pages

### 8.2 日常水質 → 別シートへの転記（GAS書き込み）

**アプリ入力データを GAS が複数のシートに書き込む**。Excel側の数式参照は使わない（月をまたいで無効になるため）。

#### 終沈3項目の書き込み先

### 8.3 月初前月最終値の自動取得（重要）

**運転管理月報 r6 J列・L列** に「前月最終金曜の返送汚泥流量読み値」を書き込む処理が必要。

#### 仕様

```
1. アプリのGoogleスプレッドシート（SS-D）には全月の入力データが連続して保存されている
2. Excel出力時、GASは前月のデータブロックから「最終金曜の返送汚泥流量読み値」を取得
3. 運転管理月報 r6 J列（No.1）/ L列（No.2）に書き込む
4. r7以降の差分は数式（K列・M列）で自動計算される
```

#### 実装ポイント

- 前月のスプレッドシートの返送汚泥流量データから「最終週ブロックの金曜（M列）の値」を取得
- 取得失敗時はテンプレの該当セルを空欄のまま（手入力でフォールバック）
- 初月（運用開始月）はデータなしなので空欄

#### コード例（GAS）

```javascript
function getPreviousMonthLastFridayValues(currentMonth) {
  // 前月のyear-month を計算
  const prevDate = new Date(currentMonth.year, currentMonth.month - 2, 1);
  const prevYear = prevDate.getFullYear();
  const prevMonth = prevDate.getMonth() + 1;
  
  // SS-Dから前月の日常水質データを取得
  const ssData = SpreadsheetApp.openById(SPREADSHEET_ID);
  const sheet = ssData.getSheetByName('日常水質_' + prevYear + '_' + prevMonth);
  if (!sheet) return { no1: null, no2: null };
  
  // 前月の最終週ブロックの金曜の値（返送汚泥流量 No.1, No.2）を取得
  // ... 各週ブロックを走査して最終データを取得
  
  return { no1: lastFridayNo1, no2: lastFridayNo2 };
}

function writePreviousMonthValuesToTemplate(templateSheet, currentMonth) {
  const values = getPreviousMonthLastFridayValues(currentMonth);
  if (values.no1 !== null) templateSheet.getRange('J6').setValue(values.no1);
  if (values.no2 !== null) templateSheet.getRange('L6').setValue(values.no2);
}
```

### 8.4 主要機器運転時間 日付セル書き込み

各週ブロックの日付セル（C列）に「令和X年X月X日 ～ 令和X年X月X日」のフルレンジ表記を書き込む。

#### 書き込み対象セル

| 週 | セル | 内容 |
|---|---|---|
| 第1週 | C5 | 令和X年X月X日（月曜）～ 令和X年X月X日（金曜）|
| 第2週 | C43 | 同様 |
| 第3週 | C81 | 同様 |
| 第4週 | C119 | 同様 |
| 第5週 | C157 | 同様 |

**注意**: 月の最終週が短い場合（例: 3月30,31日のみ）、終了日は実際の最終入力日に合わせる。

#### コード例

```javascript
function formatReiwaDateRange(startDate, endDate) {
  const reiwaYearStart = startDate.getFullYear() - 2018;
  const reiwaYearEnd = endDate.getFullYear() - 2018;
  return `令和${reiwaYearStart}年${startDate.getMonth()+1}月${startDate.getDate()}日 ～ ` +
         `令和${reiwaYearEnd}年${endDate.getMonth()+1}月${endDate.getDate()}日`;
}
```

## 9. 既知の課題（Phase 2 で対応）

### 月によって日数が異なる

現在のテンプレは3月（31日）固定。4月（30日）・2月（28-29日）の場合：
- 運転管理月報・水質管理報告: 末尾の余り行を空欄処理（GAS側で対応）
- 各シートの統計行範囲は31日分のまま（空欄は無視されるので問題なし）

### 第1週・第5週が短い月

例: 月初が金曜（第1週は1日のみ）、月末が月曜（第5週は1日のみ）。
- アプリ側で週ブロックを動的に判定（既存の加計ロジック流用可）
